攒钱狂魔 | Money Guard Pro
「攒钱狂魔」 是一款基于 Neumorphism (新拟态) 设计风格的单文件极简记账 Web App。它专注于通过“欲望减法”和“3D 实体触感”来提升记账的自律感与趣味性。

✨ 核心特性
🕹️ 3D 实体交互：全站采用新拟态 3D 按钮，拥有真实的按下凹陷与物理回弹反馈，让记账像操作实体按键一样爽快。

⚖️ 动态资产看板：顶部实时显示 “总结余 = 总收入 - 总支出”，一眼看清你的财富底牌。

🛡️ 欲望减法逻辑：特别设计的“自律勋章”功能，记录你每一次抵制住奶茶、游戏等消费诱惑省下的钱。

📊 多维度分类：内置收入、支出、抵御诱惑三种独立的分类体系，账目清晰不混乱。

📱 原生 App 体验：适配移动端，支持“添加到主屏幕”作为独立 Web App 运行，无浏览器边框，沉浸感极强。

🔒 数据安全：数据仅存储于浏览器本地 (LocalStorage)，不上传任何服务器，隐私 100% 掌握在自己手中。

🚀 快速部署 (30秒)
本项目为纯前端单文件工程，无需任何环境配置：

Fork 本仓库。

进入仓库设置 Settings > Pages。

将 Branch 设置为 main 并保存。

你的专属记账地址即刻上线！

🛠️ 使用指南
录入收入：记录你的工资、奖金等，增加你的总资产。

录入消费：选择分类记录日常开销，资产将自动扣减。

欲望减法：每当你忍住一笔不必要的消费（如忍住没喝奶茶），记入此项，作为你的“战利品”成就。

添加到桌面：在手机 Safari/Chrome 中打开 URL，选择“添加到主屏幕”，开启最佳质感体验。

🎨 UI 设计哲学
本项目深受 Skeuomorphism (拟物化) 与 Neumorphism (新拟态) 启发，通过对光影（高光与阴影）的精确计算，模拟真实物体的物理质感。

凸起阴影：6px 6px 12px #a3b1c6, -6px -6px 12px #ffffff

凹陷阴影：inset 4px 4px 8px #a3b1c6, inset -4px -4px 8px #ffffff
